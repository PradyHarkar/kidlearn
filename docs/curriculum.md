# Curriculum.md

## Purpose
This document defines the topic ladder for KidLearn.

It tells the generator, teacher, and learning engine what children should be practicing at each age band.

## Principles
- Start concrete before abstract.
- Use objects and scenes before symbols.
- Increase difficulty gradually.
- Match the local curriculum and exam style.
- Keep the content age-appropriate and meaningful.

## Country Notes
- Australia: align to Australian Curriculum and NAPLAN-style skill checks.
- UK: align to National Curriculum and KS-style progression.
- US: align to common grade-level expectations and core standards.
- India: align to NCERT-style competency learning.

## Generation Rules
- Generate questions that fit the ladder above.
- Do not jump to abstract forms too early.
- Keep the context concrete for younger years.
- Keep the wording readable for the child's age.
- Use the right visual mode for the age band.

## Teacher Review Hook
Every question should be judged against this ladder before it can go live.
